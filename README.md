# Rrenash - Albanian Bluff Card Game

An exciting implementation of the classic Albanian bluff card game, built with React, Vite, Tailwind CSS, and shadcn/ui.


## Play Online 
The fastest way to play Rrenash is straight from the web — just open the link and start a game.

https://zfabio.github.io/rrenash/

## Setup Instructions

1. Ensure you have Node.js installed.
2. Clone this repository (or copy it to your machine).
3. Install dependencies by running:
   ```sh
   npm install
   ```
4. Start the development server by running:
   ```sh
   npm run dev
   ```

## Creating a Production Build

To deploy the game on your own website, you can build the production assets:
```sh
npm run build
```

This will create an optimized build inside the `dist/` directory, which you can then upload to Vercel, Netlify, GitHub Pages, or any standard web hosting provider.
